attributed again in the lesson that uses it.

Commons has no usable equivalent for the remaining figures (problem framings, stationarity,
evaluation protocols, filter ladder, motion models/IMM, data association, resampling artifacts,
segmentation, coverage under regime change, patching, multi-modal futures) — those are generated.
