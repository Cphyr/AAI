# ML Crash Course

A 3-week onboarding course covering the core of classical and modern machine learning.
Each chapter contains short, math-first lessons, a hands-on Python walkthrough, and an assignment.
Read the lesson first, run the walkthrough second, and only then attempt the challenge.

## Suggested 3-Week Schedule

| Week | Days | Material |
| ---- | ---- | -------- |
| 1 | 1-3 | [Chapter 1 - Trees & Ensembles](chapter_01_trees_and_ensembles/main.md) |

| 2 | 1-2 | [Chapter 3 - Support Vector Machines](chapter_03_svm/main.md) |
| 2 | 3-4 | [Chapter 4 - Clustering](chapter_04_clustering/main.md) |
| 2-3 | 5, 1-2 | [Chapter 5 - Deep Learning](chapter_05_deep_learning/main.md) |
